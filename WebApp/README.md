# Whiskey-Shop
Интернет магазин по продаже различных сортов виски.

Данный проект разрабатывается как курсовая работа.

Разработка ведется с помощью технологий Java EE и Spring.

## Темы для реализации

* Simple Data
* Simple UI
* IoC
* JPA + REST
* Server-side UI
* Security
